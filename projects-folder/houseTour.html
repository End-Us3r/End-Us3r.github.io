Tour a house!

Description

This C++ program creates a text-based game where you explore different rooms in a house. Each room has a name and description, and you can move between rooms using directional commands like "north" or "east". The game starts in the living room, and as you navigate, it prints descriptions of each room you enter. After completing the tour through the house, the program concludes with a message indicating the game tour is finished.
